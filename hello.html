Les commits

git init
il fait du repertoire spécifié (par defaut le rep courant sinon) un depot git.
git status
Affiche les infos sur ce qui n'a pas encore été commit entre autres et d'autres infos.
git diff
Affiche les modif apportés à tous les fichiers si on ne precise aucun fichier à la suite. on peut ajouter k'option --cashed pour que la verification se fasse au niveau de la zone d'indexation.
git add
Ajoute les fichiers spécifiés dans la zone d'indexation. si rien est précisé il prend tous les fichiers de l'espace courant.
git commit -m "message"
Ajoute maintenant le contenu de la zone d'indexation en tant que commit.
